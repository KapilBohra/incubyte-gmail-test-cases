# incubyte-gmail-test-cases
Manual test cases for Gmail Compose feature – Traditional &amp; BDD style | Incubyte QA Assessment

## 📌 Overview
This repository contains manual test cases for the **Gmail Compose feature** as part of the 
Incubyte Test Craftsperson (Manual) hiring assessment.

The test cases cover composing and sending an email with:
- **Subject:** Incubyte
- **Body:** QA test for Incubyte

📄 Gmail_Compose_TestCases.xlsx   → Main test case document

## ✅ Testing Approach
- **Traditional Style** – Standard test cases with Preconditions, Test Steps, Test Data, and Expected Results
- **BDD Style** – Gherkin syntax using Given / When / Then format

## 🔍 Features Tested
- Opening compose window
- Entering recipient, subject, and body
- Sending email successfully
- Draft save and reopen
- CC / BCC functionality
- File attachment (valid & oversized)
- Error handling (missing fields, invalid email, network failure)
- Security (XSS in subject field)
